Security Policy
Supported Versions

This project is currently developed on the main branch.

Security fixes are provided for:

Version	Supported
Main (latest)	✅
Older releases	❌

Only the latest version available in the repository is supported.

Reporting a Vulnerability

If you discover a security vulnerability, please do not create a public GitHub issue.

Instead, report it privately by contacting the maintainer:

GitHub: @PatrickGuttmann

Please include:

A description of the vulnerability
Steps required to reproduce it
Potential impact
Any suggested mitigation or fix
What to expect
Initial acknowledgement within 7 days
Investigation and validation of the report
A security fix will be developed if the issue is confirmed
The reporter may be informed when the fix is available

Reports that are determined not to be security vulnerabilities may be closed without further action.

Thank you for helping improve the security of this project.
